
### Shape Calculator
This calculator uses a scanner to recieve input and spit out the features of a 2D or 3D shape given the dimensions.
Available shapes:
- regular polygon
-  circle
-  ellipse
-  rectangular pyramid
-  sphere
-  cylinder
-  cone
